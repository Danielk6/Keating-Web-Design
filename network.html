IP
IP stands for Internet Protocol. This is typically used with TCP (Transport Control Protocol) and is the standard network protocol used across the internet. The protocol uses messages (also known as packets), which consist of a header and the message data itself.
FTP
FTP stands for File Transfer Protocol. The networking protocol is used to transfer copies of files between two computers and is based on Internet Protocol
HTTP
HTTP stands for Hypertext Transfer Protocol. The networking protocol is a standard that web browsers and servers communicate with. It is typically used by a web browser requesting HTML files from a web server. Once the request is complete, the connection is dropped.
